Trailblazer

Trailblazer is a lightweight CSS library that provides a simple, cohesive design. You can download a copy of this library on my github page, 2themacks. Please note that I will always be aiming to improve this layout, so check out the repository for any other updates!

Here's how it works

access things by their class or element name

Here is a high level overview of you can access different elements on this framework. There are four main sections of this webpage: nav bar, header, content body, and footer.

To edit the tect of the Section Titles and Subtitles, they are the h2 and h3 elements, respectively. Modify these by selecting the h2 and h3 elements in the CSS and HTML sheets.
